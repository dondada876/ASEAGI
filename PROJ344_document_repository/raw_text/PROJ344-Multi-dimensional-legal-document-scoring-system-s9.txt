Purpose: Extract statements, classify content, detect violations

Analysis Pipeline:
Step 1: Document Classification
  └─ Identify doc type (declaration, transcript, report, etc.)
  └─ Extract metadata (date, parties, court)
  └─ Determine if under oath

Step 2: Statement Extraction
  └─ Parse into individual claims/statements
  └─ Identify speaker for each statement
  └─ Preserve context (what came before/after)

Step 3: Truth Analysis
  └─ Compare to known facts
  └─ Check against other documents
  └─ Identify contradictions
  └─ Score truth value (0-1000)

Step 4: Intent Classification
  └─ Good faith vs. bad faith
  └─ Protective vs. harmful
  └─ Score intent (0-1000)

Step 5: Context Relationship Scoring
  └─ Temporal analysis (when made relative to events)
  └─ Relational analysis (contradicts what?)
  └─ Pattern analysis (part of series?)
  └─ Score context (0-1000)

Step 6: Bad Faith Quantification
  └─ Timing manipulation detection
  └─ Forum shopping identification
  └─ Evidence concealment flagging
  └─ Score bad faith (0-1000)

Step 7: Violation Detection
  └─ Perjury elements (material + false + oath + intent)
  └─ Fraud on court (extrinsic fraud)
  └─ Child endangerment (PC 273a)
  └─ Other violations
  └─ Severity scoring (0-1000)

Step 8: Legal Weight Assessment
  └─ Proves which statutory elements?
  └─ Admissibility evaluation
  └─ Strategic value determination
  └─ Score legal weight (0-1000)

3. SCORING ENGINE
Purpose: Calculate all dimensional scores and aggregations
Dimensional Scores (0-1000 each):
# Statement-Level (Level 1)
truth_lie_score = calculate_tls(statement, evidence, contradictions)
intent_culpability = calculate_ics(statement, context, party_history)
bad_faith_quantification = calculate_bfq(timing, violations, patterns)
context_relationship = calculate_crs(temporal, relational, pattern, consequence)
evidence_quality = calculate_eqs(source, authentication, admissibility)
legal_weight = calculate_lws(element_proof, materiality, impact)

# Composite
statement_micro_score = calculate_sms(all_dimensions)

# Page-Level (Level 2)
page_score = aggregate_statements_to_page(statements)

# Document-Level (Level 3)
evidence_strength = composite(truth, verification, quality, authenticity)
legal_impact = composite(proof_level, admissibility, weight)
strategic_value = composite(case_impact, opposition_impact, context)
intent_conduct = composite(intent, bad_faith, compliance)

document_master_score = weighted_average(
    evidence_strength * 0.35,
    legal_impact * 0.35,
    strategic_value * 0.20,
    intent_conduct * 0.10
)

# Motion-Level (Level 4)
motion_strength_score = aggregate_documents_to_motion(documents)

# Party-Level (Level 5)
justice_master_score = aggregate_all_party_actions(party)
Weighting Philosophy:
Evidence reliability = most important (35%)
Legal impact = equally important (35%)
Strategic value = significant (20%)
Intent/conduct = contextual (10%)
Score Adjustments:
Violation severity multipliers

Contradiction penalties
